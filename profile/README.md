## Quick Start

```bash
# Sklonuj konkretny projekt
git clone https://github.com/wronai/<project-name>.git

# Przykład: uruchom goal (release autopilot)
pip install wronai-goal
goal --help

# Przykład: uruchom domd (dokumentacja z kodu)
pip install domd
domd scan .

# Przykład: heal (naprawa błędów shell)
pip install wronai-heal
heal
```
